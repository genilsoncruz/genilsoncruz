Projetos de Programação e DevOps

Sistema de Gerenciamento de Tarefas

Descrição: Desenvolver uma aplicação web para gerenciar tarefas e projetos. A aplicação deve permitir a criação, edição e exclusão de tarefas, além de categorização e priorização.

Tecnologias: Python (Django ou Flask), JavaScript (React ou Vue), PostgreSQL.

Funcionalidades:

  • Autenticação de usuário.
  • Interface de usuário intuitiva.
  • Sistema de notificações.

Projetos de IA

Classificador de Imagens com Redes Neurais Convolucionais

Descrição: Desenvolver um modelo de IA que classifique imagens em diferentes categorias (por exemplo, animais, objetos, etc.).

Tecnologias: Python, TensorFlow ou PyTorch, OpenCV.

Funcionalidades:

  • Treinamento do modelo com um conjunto de dados (por exemplo, CIFAR-10, ImageNet).
  • Avaliação e ajuste do modelo.
  • Interface web para upload e classificação de imagens.

Projetos de Sistemas de Tempo Real e Microcontroladores

Sistema de Controle de Tráfego

Descrição: Desenvolver um sistema de controle de tráfego que gerencie semáforos em uma interseção movimentada em tempo real.

Tecnologias: C/C++, FreeRTOS, sensores de proximidade.

Funcionalidades:

  • Controle baseado em tempo e volume de tráfego.
  • Priorização de veículos de emergência.
  • Interface para monitoramento e ajuste de parâmetros.
